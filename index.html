    StudyBlocks

    StudyBlocks is a collaborative online studying platform that allows anyone to take their flashcards anywhere. Users can take practice tests generated from their cards. StudyBlocks tracks users' scores and displays them on interactive charts, allowing users to see their progress over time. StudyBlocks also offers registration with schools and courses, allowing users access to the decks of their classmates. Together, a class can generate an enormous amount of material, making learning a team effort.

    Blake

    Blake is a modern, browser based game, written in JavaScript and inspired by the arcade classic, Snake. The development of Blake was a challenge in using only jQuery to create a highly dynamic, interactive experience. Creating a functional game was a relatively simple task, however the real challenge was creating the appearance of movement and smooth animation. Clever, dynamic manipulation of HTML elements with different CSS classes, all facilitated by jQuery, resulted in a pleasing and fun tribute to the first video game I ever played.

    This is a terminal based chess game written in ruby. It supports two human players, a human versus a computer, and even computer versus computer games.
